# 📖 每日知识

每天5个知识点，覆盖职场·生活·科普·健康·工具，自动更新。

## 在线访问

GitHub Pages 部署后，访问：
```
https://你的用户名.github.io/daily-knowledge/
```

## 本地使用

双击 `index.html` 即可打开，学习状态保存在浏览器本地。

## 每天自动更新

- 每天 **北京时间 9:00** GitHub Actions 自动运行
- DeepSeek API 生成5条新知识
- 前一日内容自动归档到「往期知识」
- 更新历史自动保留30天备份

## 技术栈

- 纯前端 HTML/CSS/JavaScript
- GitHub Actions + GitHub Pages
- DeepSeek API 生成内容
- localStorage 保存学习状态
